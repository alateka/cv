Administración y configuración de distribuciones GNU/Linux (Cliente/Servidor), así como Winodws cliente y Windows Server

Servidores:

DNS: Crear y configurar zonas en la red para asignar nombres de dominio a los equipos conectados.

DHCP: Instalar o configurar un servicio DHCP para la asignación automática de direcciones IP a los dispositivos en la red.

Proxy: Configurar un servicio proxy que puede actual tanto de cache web como toma de control de acceso a determinados sitios web

SAMBA (SMB): Para compartir directorios en una red de ordenadores Windows o una red con clientes Windows y Linux.

NFS: Exportar (Compartir) directorios a todos los ordenadores de la red mediante este protocolo.

SSH: Para la conexión remota desde una máquina cliente hacia el servidor y/o configurar SFTP para transferencia de ficheros.

Servidor de Impresión: Configurar un servicio de impresión unificada donde los clientes soliciten el uso de impresoras conectadas a una única máquina

Clientes:

SO Microsoft Windows: Instalar y/o configurar la mayoría de funciones de clientes Windows tanto para administrarlos como para optimizarlos.

SO GNU/Linux (Debian, Ubuntu, Fedora, Archlinux, CentOS, etc): Realizar instalaciones manuales de Distribuciones Linux en equipos cliente, asi como realizar configuraciones en el sistema.
